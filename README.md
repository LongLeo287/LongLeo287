## 🛡️ Security-First Architecture

OmniClaw is hardened against modern threats at every layer:

- **Supply Chain Defense** — Automated IOC scanning for compromised npm packages (e.g. axios RAT)
- **Artifact Hygiene** *(Pillar 8)* — Zero source map leaks, zero `.env` exposure in published artifacts
- **Zero-Trust Sandbox** — All external repos quarantined, gitingest-dried before touching the Brain
- **Global Blacklist** — Live IOC registry with Escape Protocol for immediate incident response
